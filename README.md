# template-svelte

Vite + SvelteKit + Svelte + TailwindCSS

## Get Started

Click **Use Template** or this [link](https://github.com/yzITI/template-svelte/generate).

Change project name in  `package.json` and `src/routes/page.svelte`
```
"name": "project-name"

<!-- you may change lang as well -->
<title>Your Title</title>
```

Install dependency

```
npm i
```

Run dev server

```
npm run dev
```
